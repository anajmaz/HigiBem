# PROJETO HIGIBEM
Esse projeto é um site para o meu projeto social que visa a acessibilidade a higiene para as mulheres em situação de vunerabilidade social.
Foi meu primeiro projeto individual de desenvolvimento.

## TECNOLOGIAS USADAS
- HTML
- CSS
- JavaScript
